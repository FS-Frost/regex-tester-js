# regex-tester-js

## Built with

- [SvelteKit](https://kit.svelte.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Bulma CSS Framework](https://bulma.io/)
- [Vite](https://vitejs.dev/)
- [bun](https://github.com/oven-sh/bun)

## Run locally

```shell
bun run dev
```

Navigate to http://localhost:5000.
